The z-index: the battle of the overlaps What happens when both classes are set to position relative? By default, the class which appears further down in the html code will get priority. However, there is a way to override this by using z-index. z-Index is a ranked priority code in CSS. You use it as such
.class of your choice {
z-index: 10;
}
