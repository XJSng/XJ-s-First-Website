How to emphasise text

Creating Bold Text
You can put text within the "strong" element to bold text
Example: here is a bold text.
You can also use the "b" element to bold text.
However, it is not advised as using CSS would not change the "b" element
Therefore, it is advise to use the "strong" tag
Creating Italic Text
You can put text within the "em" element to italicised text
Example: Here is an italicised text.
You can also use the "i" element to bold text.
However, it is not advised as using CSS would not change the "i" element
Therefore, it is advise to use the "em" tag
Alternatively, you can use the "dfn" element which stands for definition and does the same thing as "i"
