Introduction to CSS

CSS stands for Cascading Style Sheets and it is used to stylise the website

This is done by linking to a CSS file. Example <link href="style.css rel="stylesheet"/>

CSS Selectors

There are three types of selectors:

  • Type selectors
  • Descendant selectors
  • Class selectors
  • Mixing Selectors
  • Cascade
  • Box Model
  • Floats & Flex Box

Type Selectors

When we want to target an element to stylise say all paragraph text, we can use the p in CSS as such
p {
color: blue;
}

Descendant Selectors

When we want to target specific elements within another element to stylise, say the paragraph text within all header section, we can use the header p in CSS as such
header p {
color: black;
}
Think of it like a Russian doll, the computer is seaching the specific element until it finds the right element to style.

Class selectors

This paragraph has been given the class="highlight". Therefore, when we want to stylise this element all we have to do is type in .highlight in CSS as such:
.highlight {
background-color: yellow;
}

Mixing selectors

You can mix selectors like Descendant & Class Selectors to target specific class on your website. Example: You want highlight the paragraph in the footer element. You can first give the footer paragraph a "greylight" class. Then, work on the CSS file using the footer .greylight{
background-color: grey;
}

Cascade

By definition a cascade means: A small waterfall, typically one of several that fall in stages down a slope. Like a waterfall, the web browser has a default font, size and colour for their texts flowing through the website. Unlike the waterfall, we can use CSS to change the values of the website

The second thing to note is this idea of inheritance that certain CSS properties are passed along from ancestor to children elements.

The last thing to note is specificity. When two CSS properties fight over an element. For example, you want to target the text navy in this paragraph to a different colour but <p> and <span p> are fighting over this element. This is where being specific helps, not only that the ordering of our rules matter.

    The order of which CCS propertiy gets priority begining:
  1. Descendant Selector + Class Selector (footer. className)
  2. Class Selectors (.className)
  3. Deeply Nested Descendant Selectors (body footer p)
  4. Shallow Nested Descendant Selectors (footer p)
  5. Type selectors (p)

When two rules have identical specificity values, the one which comes last will be followed. Example,

p {
color: blue;
}
p {
color: grey;
}
The grey colour will be applied.

CSS Box Model

Every webpage is a collection of boxes.

All websites are a collection of boxes. We use:
  • padding to create space inside the box
  • border to create a line on the box
  • margin to create space outside the box

Floats & Flex Box

The use of floats in CSS makes columns on a website possible. Beginning with two div classes "main-area" & "sidebar" but you can name it however you see fit.
This is the "main-area" div. Next we want to create another div called "container" and move contents from the previous 2 div into the "container" div. In CSS, I am able to target the .container as so:
.container {
max-width: 940px;
margin-left: auto;
margin-right
}

Now we will create another div class called "content-area" under the "container" div. Next, we will copy and paste both "main-area" and "sidebar" div into the "content-area" div. We can now style the "content-area" div in CSS with the following:
.content-area {
border-top: 2px solid black;
border-bottom: 2px solid black;
}
We will then set both the "main-area" and "sidebar" elements to 66% and 34% respectively plus a float element as such "float:left;".
This will create 2 columns but the border will also move to the top. This is because the parent element (content-area) does not recognise floated elements.

We can fix this adding another class to the "main-area" element called "group" and it should look like this
<div class= "main-area group">
Next we will add what is called a "clearfix" method as such:
.group:before,
.group:after {
content: "";
display: table;
}

.group:after {
clear: both;
}
.group {
zoom: 1;
}
