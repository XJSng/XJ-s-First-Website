    The <div> and <span> elements

    This third article talks about the <div>, </div>, <span> & </span> elements which does not add any semantic meaning to the code but is useful for styling with CSS. The <div> element stands for "Division/Section" and is used for general purpose.

    Then there is <span> which is a non-sementic element. Basically, it can use the <span> element to stylise with CSS in a later date.

    Commenting in HTML

    This fourth article talks about how to comment on HTML. This is done by using the <!– – insert comment – –>. Comments are useful between developers working on the code to denote changes or purpose of this line of code. It is good practice to have a comment after a </div> ends.

    Sections in HTML

    Sections are like chapters of a book while articles are like individual blog post with its own self-contained chapters or sections. This also means that sections can be within another sections or sub-sections. Something to note is that heading within a section are of the same size and will look the same heading size.
