Creating a table in HTML

Using <table> element, we can create a table on this website. We follow up with the <th> element to create the header of the table. Next we will use the <tr> & <td> element to create the table row and table data respectively.

It is also best practice to put the table heading under the <thead> element and the table body in the <tbody> element. The last table row is usually labelled with the <tfoot> element.
