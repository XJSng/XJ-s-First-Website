Background Gradient

I have added a gradient background on the sidebar. This can be done by adding using the "background-image" element.
To create the current gradient from top left to bottom right, I will need to add in this code in CSS
"background-image: linear-gradient(to bottom right, #e5e9dc, #fff);"
