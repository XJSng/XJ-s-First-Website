Responsive Grids

What is a grid?

A grid is a tool that graphic designers use to make sure that their layouts follow a uniform and balanced structure. One of the most popular grids structure is the twelve column grid.

The reason the twelve column grid is so common is because twleve is easily divisble by many useful numbers. The space between the red column is called the gutter. Each gutter is 20px apart. The benefits of using grids shines when multiple rows of columns are introduced. Using a grid allows future columns to stick to the rhythm of columns making the layout look cohesive.

What is a responsive grid?

With the understanding grids, you can plan your layout for each element of your website. You can begin with a large desktop layout and trickle down to a mobile device.
