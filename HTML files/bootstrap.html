What is SASS

SASS stands for Syntactically Awesome StyleSheet, it is an extension of CSS. It uses a SASS compiler so that the computer can understand your SASS to CSS.

    There are two ways to access a SASS Compiler:
  • The easy way is to go to codepen.oi. Under CSS settings, you can select the SCSS Preprocessor
  • The professional way is to download Node.js
