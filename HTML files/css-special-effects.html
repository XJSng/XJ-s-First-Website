Introduction to CSS3 Special Effects

Box Shadows

Box shadows can be added by using the "box-shadow" element followed by the property values.
box-shadow: 2px 3px 2px rgba(0, 0, 0, 0.33);
The first two values represent the (x, y) value of the shadow, the third represents the blur radius and the forth represents the colour.

Rounded Corners

The "border-radius" element is used to create rounded corners on any boxed element. You can have up to 4 seperate value "border-radius: 20px 15px 10px 5px". You can also use the backslash element (40px/10px) to create elliptical rounding, that is to say the rounding is 40px on the horizon and 10px on the vertical.

CSS Transform

There are four ways to transform an element:

  • Rotate
  • Scale
  • Skew
  • Position (origin & placement)

Rotate

To rotate an image, we will use the "transform: rotate(15deg)" in CSS.
A negative degree would move the image anti-clockwise "transform: rotate(-15deg)"

Rotate

The rotate property used a multipler that means by using the "transform: scale(1.3)" we would transform the image x1.3. The amount of space the box uses is not more than the space it is given. It means we are only changing the scale of the element and not the margin/padding or box size of the image.

Skew

The skew property uses the X and Y axis to angle the image.

  1. The first image is skewed on the X axis "transform: skewX(15deg);"
  2. The second image is skewed on the Y axis "transfrom: skewY(15deg);"
  3. The last image is skewed on both axis "transform: skewX(15deg) skewY(15deg);"

At this point of time, we can also combine transfrom various properities as such " transform: rotate(15deg) scale(0.9) skewX(15deg);"

Position (origin & placement)

The transform-origin property to move the pivot point of our image. We can adjust the x and y axis of the image. By default the browser uses "transform-origin: 50% 50%;".

The transform position can move the image independent of other elements. This is achieved by inputing the "transform: translate(30px, 30px);" into CSS.
We can also use the position: relative property to create the exact same position. But keep in mind that position relative is used for layout purposes, while translate() is more for animation purposes


Transitions in CSS

By using the transition property(transition-property: all), we are able to create fade transition between hover effects.
